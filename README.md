## Personal Website

This repository contains the code for my personal website and the IS 492 Lab 2 assignment. All required items from the lab description are implemented and working. The site is deployed at:

https://ziyi-website.vercel.app

### Project Summary

- Built from a carefully designed prompt using v0 as the initial baseline.
- Iterated further with custom changes (half manual, half Copilot-assisted).
- Added a Playground section and refined layout/navigation logic.
- The Contact page submits data to Supabase.

### Features

- Multi-section personal website (hero, about, projects, research, publications, skills, awards, contact, playground, etc.).
- Responsive layout with smooth navigation between sections.
- Theme support and modern UI components.
- Contact form connected to Supabase for data submission.

### Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS
- shadcn/ui + Radix UI components
- Supabase (contact form submission)
